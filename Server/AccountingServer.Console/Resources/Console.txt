控制台帮助文档

财月：上月20-本月19

时间过滤器：不引起歧义的情况下，可以省略[]
[]					不作限制
[.]					今日
[..]				本周
[0]					本财月
[-1]					上个财月
[@0]				本月
[@-1]				上月
[yyyyMMdd]			某日
[yyyyMM]			某财月
[@yyyyMM]			某月
[yyyyMM yyyyMM]		某几财月
[@yyyyMM @yyyyMM]	某几月
[yyyyMMdd yyyyMMdd]	某两日之间
[yyyyMMdd ]			某日之后
[ yyyyMMdd]			某日之前
[null]				无日期


细目过滤器：
T1001				限制一级科目
T100101				限制科目
''					限制内容

借贷方向过滤器：
+					仅计入借方
-					仅计入贷方

分类汇总：
`tsc	`			按Title，SubTitle，Content分类汇总并制作报表（不含0）
``tsc	``			按Title，SubTitle，Content分类汇总并制作报表（含0）
`dtsc				按Date，Title，SubTitle，Content分类汇总（当日发生额）并制作报表（不含0）
`tscD				按Title，SubTitle，Content，Date分类汇总（当日余额，不含未变动日，零变动属于变动）并制作报表（不含0余额日）
`tscX				按Date，Title，SubTitle，Content分类汇总（当日累计发生额）并制作报表（含0累计发生额日）
`cd					按Content，Date分类汇总（当日发生额）并制作报表（不含0）
注：D和X之后不能再分类汇总


检索所有科目：
T	Titles	t		检索所有科目编号及其名称


编辑：
Alt+Enter			Upsert
Alt+Delete			Remove


Chart数据呈现：
c 0					本财月决算
c-a 0				本财月资产负债情况

生成消费报销报表：
R0		r0			本财月消费报销报表
R0-exp	r0-exp		本财月消费报销报表（用于复制到Excel）


资产：
a									检索所有资产（以报告形式显示，无计算表，不显示账面价值为零的）
a@yyyyMM							检索所有资产（以报告形式显示，无计算表，按指定日期计算账面价值，不显示账面价值为零和未购置的）
a-all								检索所有资产（以报告形式显示，无计算表）
a-li		a-list '(GUID)'				检索某资产（以报告形式显示）
a-q		a-query '(GUID)'				检索某资产
a-reg	a-register '(GUID)'			注册所有凭证
a-unr	a-unregister	 '(GUID)'		解除所有凭证
a-rd		a-redep '(GUID)'				重新计算折旧
a-reset-soft '(GUID)'				解除错误登记的凭证
a-reset-hard '(GUID)'				保留获得和处置，清除账上其它所有和某资产相关的内容
a-ap		a-apply						将本月的资产折旧/减值登记入账
a-ap		a-apply '(GUID)' [yyyyMM]	将某月的某资产折旧/减值登记入账
a-ap-co	a-apply-collapse [ yyyyMM]	将某月（含）以前的资产折旧/减值一起登记入账，按无日期处理
a-chk								检查一致性
注：此时无@为自然月，有@为财月

摊销：
o									检索所有摊销（以报告形式显示，无计算表，不显示待摊额为零的）
o@yyyyMM							检索所有摊销（以报告形式显示，无计算表，按指定日期计算待摊额，不显示待摊额为零和未开始的）
o-all								检索所有摊销（以报告形式显示，无计算表）
o-li		o-list '(GUID)'				检索某摊销（以报告形式显示）
o-q		o-query '(GUID)'				检索某摊销
o-reg	o-register '(GUID)'			注册所有凭证
o-unr	o-unregister	 '(GUID)'		解除所有凭证
o-ra		o-reamo '(GUID)'				重新计算摊销
o-reset-soft '(GUID)'				解除错误登记的凭证
o-ap		o-apply						将本月的摊销登记入账
o-ap		o-apply '(GUID)' [yyyyMM]	将某月的某摊销登记入账
o-ap-co	o-apply-collapse [ yyyyMM]	将某月（含）以前的摊销一起登记入账，按无日期处理
o-chk								检查一致性
注：此时无@为自然月，有@为财月


检查数据：
chk1				检查基本正确性（每一Voucher借贷方相等）
chk2				检查复杂正确性（每日每科目每内容，资产无贷方余额，负债无借方余额）


服务器：
lan	launch			启动
con	connect			连接
shu	shutdown		关闭
backup				备份
mob	mobile			启动/中止移动数据传输，显示/隐藏二维码


对账：
fetch				从ecard.tsinghua.edu.cn上获取学生卡交易记录，与现有账单自动对账，显示差异


其他：
?	help			帮助
exit				退出
